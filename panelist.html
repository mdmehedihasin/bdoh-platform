📖 Panelist Guidelines

Welcome, Panelist! Here's what you need to know:

  • ✅ Accuracy first. All question statements and answers must be verified before publishing. Drafts allow you to save work-in-progress.
  • 📐 LaTeX support. Use $...$ for inline math and $$...$$ for block equations.
  • 🎯 Mark the correct answer with a leading * in the Options field.
  • 📚 Difficulty scale: Easy → standard textbook, Medium → competition-level, Hard → national olympiad, Expert → international olympiad.
  • 🚫 Do not plagiarise. Always cite your source in the Source/Reference field if the question is adapted from an existing paper.
  • 🔧 Editing published questions will mark them for admin review. Significant changes require re-approval.
  • 📨 Contact an admin if you need bulk uploads or have questions about the moderation pipeline.
